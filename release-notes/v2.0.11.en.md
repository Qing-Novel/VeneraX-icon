## What's Changed

### Added
- Source migration dialog now shows linked sources at the top for quick migration without search
- Comic detail page shows linked sources and migration suggestions when loading fails
- Image favorites now supports "by comic grid" view with single-comic image grid management

### Changed
- Source migration dialog redesigned with two paths: quick migration (linked sources) and search migration (other sources)
- Linked sources displayed as collapsible cards with cover, title, author, and source name
- Migration dialog now uses radio selection for linked sources (select one to migrate directly)
- Enhanced reader performance: increased cacheExtent for continuous scroll mode and expanded prefetch window for slide mode

### Fixed
- Fixed source migration dialog: selecting search result now properly clears linked source selection
- Fixed search results expansion state lost when scrolling in migration dialog
- Fixed ExpansionTile divider lines appearing globally when expanded
- Fixed task card extra borders at top and bottom when expanded
- Fixed Android custom storage path stuck when selecting non-empty directory, added copy integrity protection
- Fixed follow updates check flooding warnings when source fails, now degrades to warning level
- Fixed read-later write failure due to missing columns in old database
- Fixed local comic directory import metadata and cover parsing failure under Android SAF

### Build / CI
- Updated disclaimer in README to explicitly cover software modification scenarios
