## What's Changed

### Added
- Draggable scrollbar on the comic detail page for quick navigation (useful when a source shows a full preview there)
- Local comic export now shows detailed progress for the packaging and folder-writing stages, so it no longer looks stuck at the end
- New guide to ignore battery optimization, reducing background tasks (follow-updates / sync / import-export) being frozen by the system
- One-tap entry to the album view from an image-favorite list item
- New toggle for whether WebDAV sync goes through the proxy
- New option to automatically clean up old reading history

### Fixed
- Fix stale data overwriting newer data and reverting your library when syncing across multiple devices
- Fix a crash from high memory use when exporting a large comic library as a merged file
- Fix a possible crash when syncing images of a large library over WebDAV
- Fix theme color and other appearance settings needing a restart to take effect after a WebDAV restore
- Fix page-turn gestures being hard to trigger when swiping quickly in the reader
- Fix a crash from accidentally triggering favorite while scrolling on the chapter comments page
- Fix the Android system back gesture occasionally going back twice and skipping the detail page
- Fix the "All" section on the downloads page not showing in-progress downloads on first open
- Fix downloads failing when the storage folder was not writable (permission is now requested as needed)
- Fix manual import/export and WebDAV image sync interfering with each other when run at the same time
