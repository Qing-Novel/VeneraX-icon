## What's Changed

### Added
- App lock now supports PIN, password, and pattern unlock, so devices without biometric hardware can still protect the app.
- Reading display settings now include a comment font size option.
- When a chapter fails to load, you can now tap to retry.

### Fixed
- Fixed a translucent/white bar appearing at the bottom of the screen on some devices.
- Fixed an occasional crash on restart after syncing.

**Full Changelog**: https://github.com/Kyosee/venera/compare/v2.1.6...v2.1.7
